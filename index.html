Kali Linux 知名資安工具介紹

專為滲透測試與數位鑑識設計的作業系統

什麼是 Kali Linux?

Kali Linux 是一個基於 Debian 的 Linux 發行版，專為數位鑑識和滲透測試而設計。它是 BackTrack Linux 的後繼者，由 Offensive Security 團隊開發和維護。Kali Linux 預裝了超過 600 個滲透測試程式，使其成為資訊安全專業人員、道德駭客和資安研究人員的首選平台。

Kali Linux 的設計理念是提供一個完整、可靠且穩定的滲透測試環境，讓使用者能夠快速進行各種資安測試與評估。無論是網絡掃描、密碼破解、無線網路檢測，還是社交工程與漏洞利用，Kali Linux 都提供了相應的專業工具。

常用資安工具介紹

N Nmap

Nmap（Network Mapper）是最流行的網絡掃描和主機探測工具之一，可以快速掃描大型網絡或單一主機，識別開放端口、運行服務及其版本。

主要功能：

  • 主機探測與存活掃描
  • 端口掃描與服務識別
  • 作業系統識別
  • 網絡拓撲測繪
  • 漏洞掃描整合（NSE腳本）
資訊收集 初學者友好

M Metasploit Framework

Metasploit 是最強大的開源滲透測試框架，包含大量已知漏洞的利用模組、輔助工具和後期開發模組。

主要功能：

  • 漏洞利用與攻擊測試
  • 後門植入與權限維持
  • 權限提升測試
  • 資料收集與網路監聽
  • 社交工程攻擊
漏洞利用 資訊收集 中級難度

W Wireshark

Wireshark 是世界上最受歡迎的網絡封包分析工具，能即時捕獲網絡流量並以人類可讀形式展示，用於網絡疑難排解與安全分析。

主要功能：

  • 深度封包檢測與分析
  • 多協議解碼與解析
  • 網絡流量實時監控
  • VoIP分析
  • 協議統計與性能分析
網絡分析 中級難度

A Aircrack-ng

Aircrack-ng 是一套完整的無線網絡安全評估工具集，用於專注於Wi-Fi安全的各個方面，特別是WEP與WPA/WPA2-PSK密鑰破解。

主要功能：

  • 監控模式封包捕獲
  • 攻擊向量：假身份驗證、非驗證等
  • WEP和WPA/WPA2-PSK密鑰破解
  • 分析無線網絡漏洞
  • 無線網絡監測與探測
無線攻擊 中級難度

J John the Ripper

John the Ripper 是一款快速且靈活的密碼破解工具，支持對多種加密哈希類型進行字典攻擊、暴力破解和規則自定義破解。

主要功能：

  • 支持多種密碼哈希格式
  • 字典攻擊與詞表生成
  • 增量模式暴力破解
  • 自定義破解規則
  • 支持分布式破解
密碼攻擊 中級難度

B Burp Suite

Burp Suite 是Web應用程序安全測試的綜合平台，提供了一套完整工具用於應用層分析和攻擊Web應用程序。

主要功能：

  • 攔截代理與流量修改
  • 自動化爬蟲與漏洞掃描
  • 請求重放與篡改測試
  • Session分析與操作
  • XSS、SQL注入等漏洞測試
Web應用安全 中級難度

H Hydra

Hydra 是一款網絡登錄破解器，支持對眾多協議進行快速字典攻擊，是滲透測試中用於暴力破解認證服務的首選工具。

主要功能：

  • 支持50多種網絡協議
  • 並行執行多任務
  • 靈活的協議模塊架構
  • 支持SSL加密服務
  • 多種輸出格式
密碼攻擊 初學者友好

Z OWASP ZAP

OWASP ZAP（Zed Attack Proxy）是一個開源的Web應用安全掃描器，可以作為代理攔截、檢查和修改應用程序的HTTP請求。

主要功能：

  • 自動化爬蟲與主動掃描
  • 被動掃描與流量分析
  • 代理攔截與流量修改
  • API滲透測試
  • 以OWASP Top 10為目標的漏洞掃描
Web應用安全 初學者友好
